## Steps to run the Repo
- Clone the repo
  ```
  git clone https://github.com/Exter-dg/Server-Performance-Stats.git
  ```
- Give the script execution permissions
  ```
  cd Server-Performance-Stats/
  chmod +x server-stats.sh
  ```
- Run the script
  ```
  ./server-stats.sh
  ```
